# Setup Guide

Place all files in repo root or docs/operating-system.

Install:
- GitHub CLI
- Claude Code
- CodeRabbit
- Node.js

Start Prompt:
'Read CLAUDE.md, PROJECT_STATE.md, AUTONOMOUS_OPERATING_SYSTEM.md and operate under the Autonomous Engineering Operating System.'

Workflow:
1. Select task
2. Implement
3. Verify
4. Open PR
5. Resolve CodeRabbit
6. Update state
7. Continue until blocked
